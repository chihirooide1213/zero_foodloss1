# ゼロ・Foodloss

## サイト概要
野菜や肉類を売りたい農家・畜産家の方々と
買いたい一般消費者を繋ぐサイト

### サイトテーマ
フードロスをゼロにする

### テーマを選んだ理由
成果物の売り先に困っている農家・畜産家の方々を助けたいため。
コロナの影響で外食産業が落ち込み、元々外食向けに出荷・販売していた層から
直接一般消費者へ売ることができるプラットフォームがあると便利と考えた。


### ターゲットユーザ
・一般消費者（主婦層）

### 主な利用シーン
・食品購入のタイミング（郵送対応）
